# vue-project

> 以vue-cli为基础，结合vue-router、vuex、axios、iview组件库搭建一个工程化前端demo

``` bash
## 下载依赖（依赖中less模块不要升级，升级会导致iview覆盖样式出错，程序无法正常运行）
yarn install

## 启动本地服务
yarn serve

## 打包生产文件
yarn build

## 检查代码格式并修复
yarn lint
```
